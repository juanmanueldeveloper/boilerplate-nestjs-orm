# File Structure

This structure help you get familiar with projects quickly.

```
.
├── app.config.ts                 
├── app.controller.ts             
├── app.module.ts                 
├── common                        
│   ├── adapters                  
│   ├── decorator                 
│   ├── exceptions                
│   ├── filters                   
│   ├── guard                     
│   ├── interceptors              
│   ├── middlewares               
│   └── pipes                     
├── constants                     
├── main.ts                       
├── modules                       
├── processors                      
│   ├── cache                       # Redis
│   ├── database                    # Mongo
│   ├── gateway                     # Socket.IO
│   ├── helper                      
│   └── logger                      # Logger
├── shared                          
│   ├── dto                        
│   ├── interface                  
│   └── model                      
├── utils                         
├── bootstrap.ts                 
└── main.ts                   

```
